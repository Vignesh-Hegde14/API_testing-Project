# API Testing Project using Postman

##  Project Overview
This project demonstrates API testing skills using **Postman** by validating CRUD operations
(GET, POST, PATCH, DELETE) on a sample Books API.

##  Tools Used
- Postman
- REST API
- JSON
- GitHub

##  Project Structure
postman/
 └── API-testing-project-vigneshh.postman_collection.json

## API Endpoints Covered
- GET /books
- POST /books
- GET /books/:id
- DELETE /books/:id

##  Environment Variable
TestURL – https://library-api.postmanlabs.com/books

##  How to Run
1. Import the collection into Postman
2. Set the TestURL environment variable
3. Run requests manually or via Collection Runner
